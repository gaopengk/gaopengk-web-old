Project Liquor

更新至2019年12月13日

这是Liquor，一个简简单单的图标包。

白色是它的外表，彩色是它的内心。

如你所愿来到了你的身边。

------------------

*此图标包基于jahirfiquitiva开源项目blueprint二次开发，GitHub：https://github.com/jahirfiquitiva/Blueprint。

*Liquor图标包图标部分遵循CC-BY(署名)-NC(非商用型使用)-SA(相同方式共享) 4.0许可。

*壁纸来自papers.co，遵循CC-BY(署名)许可。

*关于图标适配，请点击底侧的申请图标适配，选中要适配图标，点击紫色按钮，然后选择发送到电子邮箱内即可。

*适配图标统计时间：当前版本发布后的2～3天。

*发布新版本时间：debug版为当前版本发布后的第一个周五（反馈群内）；release版为当前版本发布后的第二个周五。


*喜欢白兰地。
